# 使用方法

## Step 01: 克隆仓库

``` bash
git clone https://github.com/Infinity4B/oh-my-rime.git ~/.config/ibus/rime
```

## Step 02: 复制文件

``` bash
cd ~/.config/ibus/rime
cp -r oh-my-rime/* .
```

## Step 03: Rime部署

右键点击部署

错误日志：
```bash
cat /tmp/rime.ibus.ERROR
```
